# BeginVirtualHost/EndVirtualHost directive {Directives}
Begins/Ends the VirtualHost section.\
\
Syntax: `BeginVirtualHost host[:port]` \
Syntax: `EndVirtualHost` \

\
構文: `BeginVirtualHost ホスト名[:ポート]` \
